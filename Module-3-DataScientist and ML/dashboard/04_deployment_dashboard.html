How to Test Predictions

Using Streamlit Dashboard

  1. Launch the dashboard: streamlit run app.py
  2. Navigate to the "🎯 Test Model" tab
  3. Fill in the trade details form:
    • Select message function (PREADVICE, NEWM, CANC, AMND)
    • Choose trading venue (EXCH, SECM, OTCO)
    • Select currency (USD, EUR, GBP, JPY, etc.)
    • Enter deal price
    • Set settlement and party priorities
    • Choose trade and settlement dates
  4. Click "🔮 Predict Settlement Outcome"
  5. Review the prediction results with probability scores
Note: The current interface shows example predictions. For actual model predictions, the system would need to engineer all 22 features from the input parameters using the same logic as in training.

Prediction Interpretation Guide

Understanding Probability Scores

  • High Confidence (>70%): Model is very confident in the prediction
  • Medium Confidence (50-70%): Moderate confidence, consider additional review
  • Low Confidence (<50%): Uncertain prediction, manual review recommended

Troubleshooting

Dashboard Won't Start

Issue: Streamlit fails to start
Solution:
  • Check if port 8501 is already in use
  • Verify Streamlit is installed: pip show streamlit
  • Try: streamlit run app.py --server.port 8502

Model File Not Found

Issue: Cannot load model file
Solution:
  • Run pipeline first: python3 run_pipeline.py
  • Check models/ folder exists and contains .pkl file
  • Verify file permissions

Poor Model Performance

Issue: Accuracy below expectations
Solution:
  • Check data quality report for issues
  • Ensure sufficient training data (>1000 records)
  • Review class balance in target variable
  • Consider additional feature engineering
