def main():
    """Execute the complete ML pipeline"""
    # Step 1: Data Processing
    processor = DataProcessor(data_path)
    df_clean = processor.clean_data()
    
    # Step 2: Feature Engineering
    engineer = FeatureEngineer(df_clean)
    X, y, features = engineer.prepare_for_modeling()
    
    # Step 3: Model Training
    trainer = ModelTrainer(X, y)
    results = trainer.train_all_models()
    trainer.save_model()
    
    return True

Orchestrates the entire ML workflow

Step 2: Run ML Pipeline

Execute Complete Pipeline

python3 run_pipeline.py
Pipeline Execution:
  1. Loads data from Data/xfm_trades_shaped.csv
  2. Analyzes data quality and generates report
  3. Cleans data (removes duplicates, handles nulls)
  4. Engineers 16 new features
  5. Trains 3 ML models (Logistic Regression, Random Forest, Gradient Boosting)
  6. Selects best model based on F1-score
  7. Saves model and generates reports

Troubleshooting

Common Issues:

XGBoost Import Error

Issue: XGBoost fails to load on macOS

Solution: Pipeline automatically falls back to GradientBoostingClassifier

File Not Found

Issue: Cannot find data files

Solution: Ensure you're in the project root directory

Permission Denied

Issue: Cannot write output files

Solution: Check write permissions for results/ and models/ folders
