📖 About This Demo

This is a Java Modernization Demo for Bank's Customer Service Ticketing System.

🔧 Tech Stack (Legacy)

  • Java: 8 (Legacy version)
  • Spring Boot: 2.7.18
  • Namespace: javax (pre-Jakarta EE)
  • Database: H2 in-memory
  • Security: Spring Security with basic auth

🎯 Demo Purpose

This application demonstrates:

  • Legacy Java 8 application architecture
  • Deprecated javax namespace usage
  • Security vulnerabilities (intentional for learning)
  • Code patterns that need modernization

🚀 Running the Backend

To connect this frontend to the backend API:

  1. Install Maven: brew install maven
  2. Build: cd JavaModernization && mvn clean install
  3. Run: mvn spring-boot:run
  4. Backend will be available at: http://localhost:8080
